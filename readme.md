To install -

```
cd server
npm i
mongod
npm run seeds
npm run server:dev
```

```
cd ..
cd client
npm i
npm run serve
```

You will need a Google Maps API key to run this project, which will be in the following file -

`client/private/keys.js`

and in the following format -

`export const googleMapsAPIKey = 'your_key'`
